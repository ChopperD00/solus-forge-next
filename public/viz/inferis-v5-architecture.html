▸ HERBIE v2 — Memory Architecture
⬡ HERBIE v2
Perplexity-Augmented Memory — Cross-Agent Retrieval Bus
Qdrant Vector Store
Persistent agent memory collections. Semantic retrieval across conversation history, artifacts, and session context. Agent-scoped so HAWK sees strategy history, FORGE sees architectural decisions.
Perplexity Enrichment Layer
Real-time web context injected into the vector store on retrieval. Gives HERBIE temporal awareness that pure vector stores can't provide — what happened today matters, not just what was stored last week.
Agent-Scoped Retrieval
Every agent in the fleet has read access to HERBIE. Context is filtered by domain — DESTRO surfaces market intel, PROSPECTOR surfaces prior research threads, ARCANA surfaces trend data for forecasting.
